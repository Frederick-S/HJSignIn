# 沪江打卡
## Installation
```
$ [sudo] npm install hjsignin -g
```
## Usage
1. Create a `config.json` file under `/usr/local/lib/node_modules/hjsignin/`.

    ```
    {
        "Account": "Your HuJiang user name",
        "Password": "Your password, encrypted by MD5",
        "process_cookie": 0,
        "type": 2
    }
    ```
2. Run `[sudo] hjsignin in terminal.`
